Как опубликовать это на GitHub Pages:

1) Открой репозиторий yujen2 → Add file → Upload files.
2) Залей ВСЕ файлы из этого архива в КОРЕНЬ репозитория (замени index.html).
3) Commit.
4) Settings → Pages → Source: Deploy from a branch, Branch: main, Folder: /(root) → Save.
5) Открой: https://yujenjp.github.io/yujen2/
   Редактор напрямую: https://yujenjp.github.io/yujen2/editor.html
6) На iPhone: открой editor.html в Safari → Поделиться → Добавить на экран «Домой».
